label_slugs(slug_diff=3, choke_diff=1, pre_slug_period=40)¶

Method labels the data in five different categories: slugs, first slugs, pre-slug, normal and ignore

Parameters:
  • slug_diff (int) – differential value for WHP above which a flow is considered to be a slug
  • choke_diff (int) – differential value for WH choke below which the choke is considered opened and constant
  • pre_slug_period (int) – number of minutes before first slug that is considered to be in “pre-slug” phase
