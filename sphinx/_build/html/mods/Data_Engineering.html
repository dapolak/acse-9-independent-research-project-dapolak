Data_Engineering¶

Part of slugdetection package

@author: Deirdree A Polak github: dapolak

class slugdetection.Data_Engineering.Data_Engineering(well)¶

Tools to crop and select the raw well data. Converts data from a Spark dataframe to Pandas.

Parameters:well (Spark data frame) – data frame containing the pressure, temperature and choke data from a well.
well_df¶

data frame containing all of the pressure, temperature and choke data from a well. None values have been dropped

Type:Spark data frame
well_og¶

original data frame copy, with None values

Type:Spark data frame
features¶

List of the features of the well, default “WH_P”, “DH_P”, “WH_T”, “DH_T” and “WH_choke”

Type:list of strings
thresholds¶

Dictionary with important features as keys, and their lower and upper thresholds as values. This is used for cropping out of range values. The set_thresholds method allows user to change or add values.

Type:dictionary
clean_choke(method='99')¶

Method to clean WH_choke variables values from the well_df Spark data frame attribute

Parameters:method (str (optional)) – Method to clean out WH_choke values. “99” entails suppressing all the data rows where the choke is lower than 99%. “no_choke” entails setting to None all the rows where the WH_choke value is 0 or where it is non constant i.e. differential is larger than 1 or second differential is larger than 3 (default is ‘99’).
data_range(verbose=True)¶

Ensures variables within the dataframe well_df are within range, as set by the attribute thresholds. The out of range values are replaced by the previous in range value

Parameters:verbose (bool (optional)) – whether to allow for verbose (default is True)
df_toPandas(stats=True, **kwargs)¶

Creates a copy of Spark data frame attribute well_df in Pandas format. Also calculates and stores the mean and standard deviations of each column in the Pandas data frame in the class attributes means and stds.

Parameters:
  • stats (bool (optional)) – Bool asserting whether or not to calculate means and standard deviations of each columns/variable (default is True)
  • kwargs –
    features : list of str
    feature names/ column headers to include in pandas data frame pd_df attribute
Returns:

pd_df – Pandas data frame of original well_df Spark data frame
Return type:

Pandas data frame

Simple plot function to plot the pd_df pandas data frame class attribute.

  • ax2_label (str (optional)) – Label for second axis, for non pressure features (default is “Temperature in C // Choke %”)

    features: list of str
    List of features to include in the plot
Returns:

data plot figure

Figure
