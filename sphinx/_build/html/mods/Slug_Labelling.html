class slugdetection.Slug_Labelling.Slug_Labelling(well)¶

Clusters together flow types based on behaviour of variables using Unsupervised Learning

Parameters:well (Spark Data Frame) – well data frame, includes continuous pressure and temperature data
Kmean_classification(n_labels, verbose=True)¶

Classifies the data from the feature_vec attribute in n_labels clusters.

Parameters:
  • n_labels (int) – Number of labels to sort the feature_vec data in (number of clusters)
  • verbose (bool (optional)) – Whether to print results (default is True)
feature_vector(window_size=20, step=5, standardise=True, keep_ts=False)¶

Transforms the raw data into features vectors containing the all the data points in the time window set by the window_size. The feature vectors can then be used for classification.

Parameters:
  • window_size (int (optional)) – Length of time in minutes that will be considered in the feature vectors (default is 20)
  • step (int (optional)) – Time step between each window of time. Can also be considered as the overlap in time. For example, if the step is set to 5, a feature vector will be created each five minutes of the continuous data set (default is 5)
  • standardise (bool (optional)) – Whether to standardise the data (default is True)
  • keep_ts (bool (optional)) – Whether to keep to timestamp column (default is False)
get_labels()¶

Get the labels indices of the feature vector data as clusterd by KMean_Classification method.

Returns:labels – List of labels. There will be as many different labels as the number of clusters selected for the Kmeans_classification method.
Return type:list of int

unpack_feature_vector¶

After the creation of the labels for each feature_vec, unpack the vectors into short continuous data frames of size window_size attribute for visualisation purposes.

Engineers should be able to visually identify the flows

Returns:unpacked_list – List of size n_labels. Each entry in the list is a data frame containing the data for the windows of time clustered in one cluster.
Return type:list of Pandas data frames
